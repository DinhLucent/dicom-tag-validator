> Python CLI tool for validating DICOM tags against the DICOM standard with detailed error reporting and HL7 integration support

## ✨ Features

- ✅ Python CLI tool for validating DICOM tags against the DICOM standard with detailed error reporting and HL7 integration support

## 📦 Installation

```bash
git clone https://github.com/DinhLucent/dicom-tag-validator.git
cd dicom-tag-validator
pip install -r requirements.txt
```

## 🚀 Quick Start

```bash
python src/main.py
```
